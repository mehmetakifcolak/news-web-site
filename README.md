# News Web Site

Bu proje, ücretsiz bir haber API'si kullanılarak geliştirilmiş basit bir
haber web sitesidir.\
Tamamen HTML, CSS ve JavaScript kullanılarak hazırlanmıştır. Herhangi
bir framework veya kütüphane kullanılmamıştır.

## Amaç

JavaScript ile API'den veri çekme, gelen veriyi işleme ve dinamik olarak
ekrana basma pratiği yapmak amacıyla geliştirilmiştir.

## Özellikler

-   API üzerinden güncel haberleri çekme
-   Haberleri dinamik olarak listeleme
-   Responsive (mobil uyumlu) tasarım
-   Basit ve temiz arayüz
-   Framework kullanılmadan geliştirilmiş yapı

## Kullanılan Teknolojiler

-   HTML5
-   CSS3
-   JavaScript (ES6)
-   Free News API

## Proje Yapısı

news-web-site/ │ ├── index.html ├── style.css ├── script.js └──
README.md

## Nasıl Çalıştırılır?

1.  Repoyu klonlayın: git clone
    https://github.com/mehmetakifcolak/news-web-site.git

2.  `index.html` dosyasını tarayıcıda açın.

Ekstra bir kurulum gerektirmez.

## Not

Kullanılan API'ye göre bir API anahtarı (API Key) gerekebilir.
Gerekliyse `script.js` içerisinden kendi anahtarınızı ekleyebilirsiniz.
